üöÄ Getting Started

Project Structure

This project utilizes two key submodules:

  • LVGL: A powerful graphics library.
  • LVGL ESP32 Drivers: Hardware drivers for LVGL. (Note: A custom patch for version 8.3 is provided).

Build and Run

  1. Clone the repository:
    git clone --recursive https://git.byte-lab.com/byte-lab/academy/blesa/blesa-2024/majda-bakmaz/mashina.git
  2. Apply the LVGL 8.3 patch:
    cd components/lvgl_esp32_drivers
    git apply --ignore-space-change --ignore-whitespace ../lvgl_esp32_drivers_8-3.patch
  3. Set up ESP-IDF environment:
    . $HOME/esp/esp-idf/export.sh
  4. Build the project:
    idf.py build
  5. Flash the project to your ESP32:
    idf.py flash /dev/ttyUSB0
    ‚ö†Ô∏è Ensure the TCH_IRQ switch is set to OFF if using the ByteLab Development Kit.
  6. Monitor the output:
    idf.py monitor /dev/ttyUSB0
